Aplicación web para gestión de gastos personales desarrollada con Next.js, TypeScript y Prisma.

Funcionalidades
Registro de gastos
Filtros por fecha
Dashboard financiero
Gráficos interactivos
Gestión de categorías
Edición y eliminación de gastos
Resumen financiero automático

Tecnologías
Next.js
TypeScript
Prisma ORM
PostgreSQL / SQLite
Recharts
CSS Modules
